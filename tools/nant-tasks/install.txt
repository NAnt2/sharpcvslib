Install:
--------

    1) copy the format task binary into your nant install bin directory
    2) create a task that uses the astyle formatter:

        <project name="test-astyle" default="style">
            <property name="base.dir" value="e:/test/testing-astyle" />
            <property name="src.dir" value="${base.dir}/src" />

            <target name="style">
                <astyle program="${base.dir}tools/astyle/astyle.exe"
                        options-file="${base.dir}/tools/astyle-options.astylerc">
                    <sources>
                        <includes name="${src.dir}/**.cs" />
                    </sources>
                </astyle>  
                <!-- OPTIONAL!!!  Remove all of the "orig" files from the directory.  
                    You may want to compile your project first before you do this. 
               <delete>
                    <fileset>
                        <includes name="${src.dir}/**.orig"/>
                    </fileset>
                </delete>
                -->
            </target>    
        </project>
        
     3) execute the task by running nant.
     4) Optionally verify the sources and remove the original files (specified by .orig).
