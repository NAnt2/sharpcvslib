cvsnt:
------
cvsnt is used to supply a cvs server with a predefined
set of files for easy running of the test suite. The cvsnt
binaries should only be used for tests, keep them separate
from the sharpcvslib in any other respect.

cvsnt version 2.0.34 download URL:
http://www.cvsnt.org/archive/cvsnt-2.0.34-bin.zip
Unzipped into tools/cvsnt/Win32
All credits to the cvsnt team for a great product.
