This folder contains a pre-built binary for NAnt on Linux.  The binary was built with the following versions:
	mono - 0.29 RPM
	NAnt - 0.84 release
	NAntContrib - 0.84 release; only the VersionTask.dll file compiled (custome build)
	
	O/S - Fedora Core 1 (Upgraded from RedHat Linux 9)
	
	The build.sh wrapper calls these binaries.
